## 🎯 Overview

**HEBINT** (pronounced "Heh-bint", short for **He**ba **B**eirut **INT**elligence) is a powerful OSINT (Open Source Intelligence) tool that automates the process of searching for usernames, emails, and phone numbers across **150+ platforms**. It's designed for security researchers, investigators, cybersecurity professionals, and anyone interested in digital footprint analysis.

### Why HEBINT Stands Out:
- ✅ **150+ platforms** - From social media to coding sites, gaming platforms to Arabic forums
- ✅ **Arabic platform support** - **UNIQUE FEATURE!** Searches Arabic sites other tools miss
- ✅ **Beautiful colored output** - Easy to read, professional presentation
- ✅ **JSON export** - Save results for later analysis, reporting, or evidence
- ✅ **Concurrent checking** - Fast multi-threaded searches (20+ simultaneous checks)
- ✅ **Always visible Facebook links** - Even when profiles don't exist, get search links
- ✅ **Self-contained** - Virtual environment keeps dependencies isolated
- ✅ **Cross-platform** - Works on Linux, macOS, and Windows
